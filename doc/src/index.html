First installation

To get started, you need to do the following:

  1. Make sure you have PHP installed. If you don't, visit the PHP how to install page and install it.
    Your PHP version must be >= 7.1.
  2. Make sure you have Node installed. If you don't, visit the Node download page and install it.
    Your Node version must be >= 10.15.3.
  3. Make sure you have Composer installed. If you don't, visit the Composer download page and install it.
    Your Composer version must be >= 1.8.5.
  4. Go into client/ and copy .env.example to .env in the same location. Replace all field as necessary.
  5. Into client/ , run npm install and composer install to fetch all dependencies from .lock files.
  6. The next thing you should do is set your application key to a random string with php artisan key:generatein client/. Typically, this string should be 32 characters long and set to APP_KEY in your .env file.
  7. Finally, you may need to grant the web server write permissions to the app/storage directories.
  8. You are set!
